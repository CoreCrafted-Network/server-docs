==============
Games Server
==============

**The Games Server** , as it's name explains, it is an
integrated server with various of games. The game it provide
include popular pvp / common games, original games [#]_ , and
arcade mini-games.

Since the second re-build of the server, the games servers are integrated into the Central Server (Hub) and are directly accessible from the hub through portals and server selection menu

.. [#] More accurately, crazy games made with ThisTNTSquid's creativity, and are grouped in the Creative servers selection menu
